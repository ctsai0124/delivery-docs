# 送達文件產製系統

已退公立學校教職員退休所得重行審定——送達證書／簽收單／信封套印產製工具。

## 特性
- 純靜態單一 HTML（`index.html`），無後端、無資料庫
- 所有 Excel 解析與文件產製皆在使用者瀏覽器內完成，**名冊個資不會上傳伺服器**
- 程式庫已全部內嵌，無外部 CDN 依賴，內網環境亦可使用

## 使用流程
1. 下載 Excel 範本 → 2. 填妥後上傳 → 3. 勾選人員 → 4. 勾選文件種類，輸出 Word 或列印／PDF

## 部署
任何靜態網頁空間皆可（GitHub Pages、Zeabur Static、Nginx…），
僅需將 `index.html` 放至網站根目錄。
